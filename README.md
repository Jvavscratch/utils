# @jvavscratch/utils

Utilities module for Jvavscratch project, providing common helper functions and tools.

## Features

- Block utilities
- Build utilities
- File system operations
- Error handling
- Evaluation tools

## Installation

```bash
npm install @jvavscratch/utils
```

## Usage

```javascript
const utils = require('@jvavscratch/utils');

// Use utility functions
```
